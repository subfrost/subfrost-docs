# SUBFROST APP Overview

*The SUBFROST APP is currently in development but will be used to showcase the best possible UX on Bitcoin L1.*

## Key Features

- **SWAP**: Single-transaction AMM swaps between BTC, ZEC, SOL, ETH and Stables for the first time ever
- **LIQUIDITY POOLS**: Provide liquidity for the above currencies and earn fees + incentives
- **FUTURES MARKET**: Bitcoin futures contracts offered on mining rewards
- **YIELD VAULTS**: Automated DeFi strategies to earn yield on native BTC and high-volume tokens

## Superior User Experience

- Use native BTC directly; wrapping/unwrapping abstracted away like in top experiences found on Ethereum and Solana
- Native Bitcoin L1 settlement for all trades
- Stablecoin on-ramp via bUSD/USDT support
- And much more...
