# Glossary

Comprehensive glossary of SUBFROST terminology.

## A

**Alkanes** - WASM-based smart contract metaprotocol on Bitcoin L1.

**AlkaneId** - Unique identifier for Alkanes contracts: `{block, tx}` tuple.

**API** - Application Programming Interface for querying SUBFROST services.

## C

**CGGMP24** - Threshold ECDSA signature scheme for Ethereum and ECDSA chains.

**Cellpack** - Simplified message format: `[opcode, block, tx, ...args]`.

**Circuit Relay** - libp2p protocol enabling NAT traversal.

## D

**DCUtR** - Direct Connection Upgrade through Relay, hole punching protocol.

**DKG** - Distributed Key Generation for threshold signatures.

**dxBTC** - Yield-bearing Bitcoin vault tokens.

## E

**Esplora** - Electrs-compatible blockchain query API.

## F

**FROST** - Flexible Round-Optimized Schnorr Threshold signatures.

**frBTC** - Synthetic Bitcoin token on Alkanes [32:0].

**ftrBTC** - Future Bitcoin options contracts [31:0].

**Fuel** - Computational budget for WASM execution (DoS protection).

## G

**GossipSub** - libp2p publish-subscribe protocol for message propagation.

**Gauge** - Liquidity mining contract with boost mechanics.

## K

**Kademlia DHT** - Distributed hash table for peer discovery.

## L

**Layer-0 (L0)** - Consensus layer above all blockchains for cross-chain coordination.

**Lua Scripting** - Programmable API queries using Lua language.

## M

**Metaprotocol** - State machine derived from blockchain data.

**Metashrew** - WASM indexer runtime for Bitcoin metaprotocols.

## N

**Noise Protocol** - Cryptographic framework for encrypted P2P communications.

## O

**Opcode** - Operation code identifying contract functions.

**Ord** - Ordinals metaprotocol for Bitcoin inscriptions and NFTs.

## P

**Protoburn** - Mechanism to burn runes and create protorunes.

**Protomessage** - OP_RETURN output with protocol messages.

**Protorunes** - Runes burned into subprotocol-only assets.

**Protostone** - Extended Runestone with Protocol field for protorunes.

**PSBT** - Partially Signed Bitcoin Transaction.

## R

**Runestone** - Data structure for runes protocol messages.

**Runes** - Fungible token protocol on Bitcoin.

## S

**Signal** - State trigger in Layer-0 consensus programs.

**SubP2P** - SUBFROST's P2P networking layer (libp2p + WebTransport).

**SubRelay** - Circuit relay v2 server.

**SubProxy** - SOCKS5/HTTP proxy for P2P.

**SubTun** - VPN tunnel over P2P.

## T

**Threshold Signature** - Cryptographic signature requiring t-of-n parties.

## U

**UTXO** - Unspent Transaction Output.

## V

**vfsize** - Virtual fuel size for fuel allocation.

**ve-token** - Vote-escrowed token (time-locked for governance).

## W

**WASI-P2** - WebAssembly System Interface Preview 2.

**WASM** - WebAssembly bytecode format.

**WebTransport** - Modern web transport protocol (QUIC over HTTP/3).
