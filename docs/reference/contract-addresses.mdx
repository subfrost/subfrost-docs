## Regtest Deployment

Use deployment scripts:

```bash
cd alkanes-rs/scripts
./deploy-regtest.sh
```

Default addresses:
- frBTC: [32, 0]
- ftrBTC: [31, 0]
- yvfrBTC: [4, n] (check script output)
- dxnormBTC: [4, n+1] (check script output)

## Verification

Verify contract bytecode:

```bash
alkanes alkanes getbytecode \
  --alkane-id "`<block>`:`<tx>`" \
  -p mainnet
```
