# Network Information

SUBFROST network details and endpoints.

## Mainnet

**Bitcoin Network**: Bitcoin Mainnet  
**Genesis Block**: 880000  
**Protocol ID**: 1 (Alkanes)

### Endpoints

| Service | URL |
|---------|-----|
| JSON-RPC | `https://mainnet.subfrost.io/v4/jsonrpc` |
| REST API | `https://mainnet.subfrost.io/v4/api` |
| Relay | `https://p2p.subfrost.io` |
| WebSocket | `wss://mainnet.subfrost.io/v4/ws` |

### Explorer

- [Alkanes Explorer](https://explorer.subfrost.io)

## Regtest (Local Development)

**Bitcoin Network**: Bitcoin Regtest

### Docker Compose

```bash
git clone https://github.com/kungfuflex/alkanes-rs
cd alkanes-rs
docker-compose up
```

**Services:**
- Bitcoin Core: localhost:18443
- Indexer: localhost:8080
- JSON-RPC: localhost:8080

## Status

Check network status:

```bash
# Mainnet
curl https://mainnet.subfrost.io/v4/api/health

# Signet
curl https://signet.subfrost.io/v4/api/health
```
