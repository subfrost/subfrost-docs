# Getting Started

Get started with SUBFROST development in minutes.

## Prerequisites

- Rust 1.70+
- Bitcoin Core (optional, for local testing)
- Git

## Installation

### Install alkanes-cli

```bash
git clone https://github.com/kungfuflex/alkanes-rs
cd alkanes-rs
cargo build --release

# Binary at: target/release/alkanes
export PATH=$PATH:$(pwd)/target/release
```

### Install subfrost-cli

```bash
git clone https://github.com/subfrost/subfrost
cd subfrost
cargo build --release -p subfrost-cli

# Binary at: target/release/subfrost-cli
export PATH=$PATH:$(pwd)/target/release
```

## Quick Start

### 1. Create Wallet

```bash
alkanes wallet create
alkanes wallet addresses
```

### 2. Get Testnet Coins

Use a signet faucet or run regtest locally.

### 3. Wrap BTC

```bash
alkanes alkanes wrap-btc 100000 \
  --from "p2tr:0" \
  -p signet \
  -y
```

### 4. Check Balance

```bash
alkanes alkanes getbalance \
  --address "bc1p..." \
  --alkane-id "32:0" \
  -p signet
```

## Development Environment

### Docker Compose (Recommended)

```bash
git clone https://github.com/kungfuflex/alkanes-rs
cd alkanes-rs
docker-compose up
```

Includes:
- Bitcoin Core (regtest)
- Alkanes indexer
- JSON-RPC server

### Local Setup

```bash
# Start Bitcoin Core
bitcoind -regtest -daemon

# Start indexer
metashrew --indexer alkanes.wasm \
  --start-block 0 \
  --daemon-rpc-url http://localhost:18443

# Start JSON-RPC server
alkanes-jsonrpc --port 8080
```
