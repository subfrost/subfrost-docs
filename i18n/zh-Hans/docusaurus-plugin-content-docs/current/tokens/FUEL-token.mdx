# FUEL 代币

:::danger[新名称，同一代币]
"FUEL" 是我们尚未发布的协议代币，此前被称为 "FROST" 代币。
:::

`FUEL` 代币是 SUBFROST 的治理和实用代币。它旨在协调网络中所有参与者的激励，并捕获 SUBFROST 协议创造的大部分价值。

*FUEL 代币经济学尚未准备好公开发布，但以下信息提供了其效用的相关见解。*

在[**此处**](../key-components/proof-of-stake)了解更多关于 SUBFROST 权益证明的信息。

## 治理

`FUEL` 代币持有者是 SUBFROST 系统的治理者。他们负责：

*   **管理 `dxBTC` 金库：** `FUEL` 持有者投票决定将 `dxBTC` alkane 中持有的 `frBTC` 部署到哪些收益生成策略中。
*   **设定费用：** `FUEL` 持有者可以投票调整将 BTC 封装/解封装为 `frBTC` 的费用以及网络上其他服务的费用。
*   **升级协议：** `FUEL` 持有者可以对升级 SUBFROST 协议及其各组件的提案进行投票。

这种去中心化治理模型确保 SUBFROST 协议始终与其用户和利益相关者的利益保持一致。

## 价值积累

`FUEL` 代币旨在从 SUBFROST 协议上的经济活动中积累价值。价值积累的主要来源是每次封装和解封装**原生 BTC** 与 `frBTC` 时产生的费用，以及未来的多项其他收入来源。

目前，`wrap` 和 `unwrap` 操作收取 0.1% 的小额费用。

这些费用的一部分用于从公开市场回购 `FUEL` 代币并销毁，对代币供应产生通缩压力。

此外，还将有时间锁定的质押来激励流动性。持有者可以在我们应用的 gauge 中质押 FUEL/frBTC LP 以提升 dxBTC 收益。这个飞轮确保了对 FUEL 价格和市场健康的正向压力。
