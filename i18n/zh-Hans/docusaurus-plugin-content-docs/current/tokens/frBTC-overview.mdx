# `frBTC` 概述
## Bitcoin L1 上可互操作的封装 BTC

`frBTC` 是与 BTC 1:1 锚定的 Bitcoin DeFi 兼容资产，可在 Bitcoin L1 上的所有 DeFi 生态系统中互操作。

这是通过一个信任最小化的流程实现的，该流程在 Bitcoin 主网（Layer-1）上锁定 BTC，并为用户**原子性地**铸造相应数量的 `frBTC`。

SUBFROST 于 2025 年第四季度在 Bitcoin 主网上线。Alkanes（10 月）和 BRC2.0（12 月）。

## 我如何使用 frBTC？

:::tip 首次使用
用户可以通过**一次**签名操作，使用原生 BTC 进入 Bitcoin L1 上的高级 DeFi 应用，完全无需预先封装 BTC。
:::
