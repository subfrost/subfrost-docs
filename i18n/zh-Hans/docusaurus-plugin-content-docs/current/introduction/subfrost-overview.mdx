# SUBFROST 概述
比特币 DeFi 代表着金融领域最具前景的前沿方向之一。然而，原生 BTC 并不直接兼容 Bitcoin L1 生态系统上的高级应用，包括 AMM、借贷协议或 zk 混币器等隐私解决方案。

SUBFROST 通过发行合成资产来释放这一潜力，使原生 BTC 能够无缝驱动下一代 DeFi。


## 什么是 SUBFROST？
SUBFROST 是一个去中心化托管协议，能够在 Bitcoin L1 上实现无需信任的 DeFi 生态系统。

它作为 [**Layer-0**](/introduction/technical-overview) 运行，通过 **Alkanes、Arch、BRC 2.0** 和 **MIDL** 等执行层直接在比特币上发行合成资产。

如果没有它，"比特币 DeFi"将继续依赖于需要信任的跨链替代方案，而我们根本不认为那是真正的"比特币 DeFi"。
