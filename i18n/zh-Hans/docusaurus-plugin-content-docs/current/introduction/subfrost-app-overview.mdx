# SUBFROST APP 概述

*SUBFROST APP 目前正在开发中，未来将用于展示 Bitcoin L1 上最佳的用户体验。*

## 主要功能

- **SWAP**：首次实现 BTC、ZEC、SOL、ETH 和稳定币之间的单笔交易 AMM 兑换
- **LIQUIDITY POOLS**：为上述币种提供流动性，赚取手续费和激励奖励
- **FUTURES MARKET**：基于挖矿奖励的比特币期货合约
- **YIELD VAULTS**：自动化 DeFi 策略，在原生 BTC 和高交易量代币上赚取收益

## 卓越的用户体验

- 直接使用原生 BTC；封装/解封装过程被抽象化，如同 Ethereum 和 Solana 上的顶级体验
- 所有交易均在原生 Bitcoin L1 上结算
- 通过 bUSD/USDT 支持的稳定币入金通道
- 以及更多功能...
