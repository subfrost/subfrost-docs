# BTCFi 兑换

*SUBFROST APP 目前正在开发中，其界面可能会发生变化。*

兑换功能允许您在支持的资产之间进行单笔交易，直接在 Bitcoin L1 上结算。无需桥接、无需手动管理包装代币——只需简单直接的交易。

**功能亮点（部分功能正在开发中）：**
- 在 Bitcoin L1 上结算的单笔交易兑换
- 自动路由以获取最优汇率
- 支持 Alkanes 和 BRC20s 的跨 BTCFi 协议交易
- 支持 USDT、SOL、ZEC 等跨链资产
- 与原生 BTC 直接互换

![Main Swap Interface](/img/app/MainSwapInterface.png)

## 如何兑换

### 第 1 步：选择代币

点击代币选择器，选择您要兑换的**来源**代币（您发送）和您要接收的代币（您接收）。

![Token Selector](/img/app/TokenSelectorPopup.png)

### 第 2 步：输入金额

输入您想要兑换的金额。使用快速选择按钮方便操作：
- **25%** / **50%** / **75%** / **Max**（最大值）

余额条通过颜色编码的反馈显示您的使用情况。

![Balance Bar](/img/app/BalanceBar.png)

### 第 3 步：自定义设置和审核

在确认之前，您将看到：
- **兑换汇率** — 每单位可获得多少代币
- **兑换路径** — 您的兑换所经过的路径（可能显示中间代币）
- **最低接收量** — 滑点后的保证最低数量
- **截止期限（区块数）** — 报价过期前的区块数
- **滑点容差** — 可接受的最大价格波动
- **矿工费率** — 交易的网络费用

![Quote Details](/img/app/QuoteDetailsSwapSummary.png)

### 第 4 步：确认并执行

点击"Swap"发起交易。系统将提示您使用已连接的钱包进行签名。

## 兑换路径

AMM 会自动为您的兑换选择最优汇率的路径：

| 路径类型 | 描述 |
|----------|------|
| **直接兑换** | Token A ↔ Token B（两者在同一池中） |
| **多跳兑换** | Token A → 桥接代币 → Token B |
| **包装/解包** | BTC ↔ frBTC（1:1 转换） |

### 直接 BTC 兑换

当从 **BTC 兑换为其他代币**时，应用会先将您的 BTC 包装为 frBTC，然后兑换为目标代币。这一过程是原子性的，用户无需手动包装 BTC。包装过程完全被抽象化处理。

当**兑换为 BTC** 时，过程相反——您的代币先兑换为 frBTC，然后自动解包为原生 BTC。在 SUBFROST 协议发送解包后的 BTC 之前，有一个 3 个区块的确认期（安全机制）。

## 跨链兑换

对于跨链代币（USDT、USDC、ETH、SOL、ZEC），界面将引导您完成简单的桥接流程：

1. **用户发送** — 等待您的存款
2. **确认中** — 在源链上确认
3. **兑换执行** — 转换和兑换
4. **SUBFROST 发送** — 用户收到资金
5. **完成** — 资金已确认

用户执行第 1 步后，后续步骤将自动完成。

## 流动性池

切换到**流动性**选项卡以提供或移除流动性：

![Liquidity Pool Interface](/img/app/LiquidityPoolInterface.png)

### 提供流动性

1. 选择 **Add**（添加）选项卡
2. 选择您要提供的代币对
3. 输入两种代币的金额
4. 点击"Add Liquidity"并签署交易

流动性提供者从其所在池的交易中赚取手续费。

注意：对于 BTC 池，用户的 BTC 会被原子性地包装为 frBTC。

### 移除流动性

切换到 **Remove**（移除）选项卡以提取您的流动性头寸并取回您的代币。

## 使用技巧

- 如果您是平台新手，**从小额开始**以了解操作流程
- **关注大额交易的价格影响**；将交易拆分为多笔兑换可能获得更优汇率
