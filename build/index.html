Welcome to the SUBFROST Documentation

SUBFROST is a novel trust-minimized Bitcoin sidechain that leverages the FROST (Flexible Round-Optimized Schnorr Threshold) signature scheme to secure its two-way peg. It introduces a Proof-of-Stake (PoS) consensus mechanism, where validators stake the native FROST token to secure the network and earn rewards.

This documentation provides a comprehensive guide to understanding, using, and developing on the SUBFROST platform.
